# Research Notes

This directory contains public-safe research notes derived from experiments, comparisons, and integration studies.

## Rules

- Use `templates/research-report-template.md` for new notes.
- Keep raw local drafts out of the repository when they contain private paths, private endpoints, credentials, or environment-specific details.
- State the research date and scope.
- Separate observation from recommendation.
- Mark claims that need later verification.
- Add every durable note to `registry/resources.json`.
