## 维护规则

- 英文文档为准。
- 中文翻译应简洁、准确，不添加英文原文没有的私有部署细节。
- 不翻译、传播或补充任何密钥、cookie、session、私有 endpoint、本机绝对路径或个人邮箱。
- 新增长期可用中文资源时，同步更新 `registry/resources.json`。
