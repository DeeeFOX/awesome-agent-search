# Social Preview

Use [assets/social-preview.png](../assets/social-preview.png) as the repository social preview image.

The editable source is [assets/social-preview.svg](../assets/social-preview.svg).

## Recommended Export

- size: `1280x640`
- format: PNG
- title: `Awesome Agent Search`
- subtitle: `Search integration research for coding agents`

## GitHub Setup

GitHub social preview images are configured in repository settings.

1. Open repository settings.
2. Go to social preview.
3. Upload `assets/social-preview.png`.

## Privacy Notes

- The artwork contains only public repository identity.
- It does not include personal email addresses, private endpoints, local paths, credentials, or deployment details.
